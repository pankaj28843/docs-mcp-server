# User Guide

This user guide covers advanced topics for the CI test documentation.

## Configuration

Configure your application with these settings:

- `debug`: Enable debug mode
- `timeout`: Set request timeout
- `retries`: Number of retry attempts

## Advanced Features

### Search Integration

The search feature supports:
- Full-text search
- Fuzzy matching
- Boolean operators
- Field-specific queries

### Browse Navigation

Navigate through documentation using:
- Directory tree browsing
- Hierarchical structure
- File system integration
